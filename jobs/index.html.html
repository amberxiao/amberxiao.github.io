Interns Wanted

We are looking for smart motivated students to help us build great software that powers our immune repertoire data analysis. You don't need to wear lab coat, but you need to be able work legally in US. Email your inquiry to jobs@immduxllc.com

Core Platform

You will work on our distributed data storage and computation on top of AWS cloud, managing and optimizing resource allocation, and help develop models and algorithms for the distributed execution of analysis pipelines and tools. An ideal candidate would have...

  • hands-on experience in AWS technologies such as EC2, Lambda, S3, Auto Scaling, API Gateway etc
  • expert level *nix skills
  • scripting skills (Node.js, Python, Perl, etc.) as well as Java programming

Data Visualization

You will design and develop explorable, interactive visualization of complex ulti-dimensional data that is critical to scientific discovery and clinical decision-making. Important insight into patterns that underlie the data is the goal here. An ideal candidate would have...

  • hands-on experience with visualization tools in javascript (e.g. D3) to display very large data sets interactively
  • familiarity with plotting in R or Python is a plus
  • experience with Javascript frameworks such as Angular, or React would be a bonus
