Interns Wanted

We are looking for smart motivated students to help us build great software that powers our immune repertoire data analysis system. You don't need to wear lab coat and we have flexible hours, but you need to be able work legally in US. Email your inquiry to ut.jobs@immudx.com

Core Platform

You will work on our distributed data storage and computation system on top of AWS cloud, managing and optimizing resource allocation, and helping develop models and algorithms for the distributed execution of bioinformatic analysis pipelines and tools. An ideal candidate would have...

  • hands-on experience in AWS technologies such as Lambda, CloudFormation, Batch, Auto Scaling, API Gateway etc
  • expert level *nix skills
  • scripting skills (Node.js, Python, Perl, etc.) as well as Java programming

Data Visualization

You will design and develop explorable, interactive visualization of complex multi-dimensional data that is critical to scientific discovery and clinical decision-making. Important insight into patterns that underlie the data is the goal here. An ideal candidate would have...

  • hands-on experience with visualization libraries in javascript (e.g. D3) to display very large data sets interactively
  • familiarity with plotting in R or Python is a plus
  • experience with Javascript frameworks such as Angular or React is desirable

Graphic Design

You will design high quality graphics for logos, brochures, signage and other branding and marketing print material. You will also enhance website visuals with beautiful icon, layout, infographic and interaction designs.

  • no coding experience is required
  • knowledge of layout framework such Bootstrap is a plus
  • experience with Adobe Creative Suite and other design tools is a must.
