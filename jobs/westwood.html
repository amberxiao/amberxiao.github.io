Interns Wanted

We are looking for smart motivated students to help us build great software that powers our immune repertoire data analysis system. You don't need to wear lab coat, but you need to be curious and diligent. Email your inquiry to westwood.intern@immduxllc.com

Backend Platform

You will work on our distributed data storage and computation on top of AWS cloud and help develop models and algorithms for the distributed execution of analysis tools. An ideal candidate would have...

  • experience in AWS technologies such as EC2, S3, Lambda
  • Unix command line familiarity
  • scripting skills (Node.js, Python, Perl, etc.) as well as Java programming

Data Visualization

You will design and develop explorable, interactive visualization of complex ulti-dimensional data that is critical to scientific discovery and clinical decision-making. An ideal candidate would have...

  • experience with visualization tools in javascript (e.g. D3)
  • familiarity with plotting in R or Python is a plus
  • experience with Javascript frameworks such as Angular or React

Web Design

You will design and build highly usable and pleasing web site and web application UI. Logos and print material design and web content editing are also part of your job. An ideal candidate would have ...

  • ability to write clean HTML and CSS code by hand
  • knowledge of Bootstrap framework and javascript
  • experience with Photoshop, Illustrator and other interaction prototyping tools
